This asset includes mesh synchronize script named "Equipment Sync".
It provides you possibility to fully syncronize the equipment that uses Skinned Mesh Renderer.

WARNING!
THIS ASSET DOES NOT INCLUDE INVENTORY/EQUIPMENT SYSTEM OR THIRD PERSON CONTROLLER!

To use this script, you have to add the component name "Equipment Sync". Set sizes both for Meshes and Materials.
Element 0 of Materials will set the material for Element 0 of Meshes.

For example:

Element 0(Meshes) - Peasant Pants;
Element 0(Materials) should contain material for Peasant Pants
